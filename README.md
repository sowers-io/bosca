Bosca
===

Bosca is a Content Management System.

### Getting Started

Run

```bash
./scripts/start-services
./scripts/run-release
```

GraphQL: http://127.0.0.1:8000/

Username: `admin` Password: `password`

The admin interface and the workflow runner infrastructure will be released soon.
